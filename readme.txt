В данном проекте есть два примера реализации - файл main_memory (когда данные, хранимые в HistoryMessagesMemoryStorage, хранятся в базе данных) и файл main_database, в котором показан пример, когда данные, хранимые в классе HistoryMessagesDatabaseStorage, хранятся в базе данных. Классы, относящиеся к пользователю, хранятся в модуле Users_module, классыб относящиеся к истории сообщений, хранятся в History_module. Показзаны примеры работы с абстрактными классами. применение статических методов, наследование, полиморфизм и доступ(чтение,запись,удаление) в базу данных SQLlite.
